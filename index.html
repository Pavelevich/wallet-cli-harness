Codex plugin

Wallet CLI Harness

A small JSON harness for Codex agents that run @ledgerhq/wallet-cli: strict flag checks, final-object result parsing, and the same wallet-cli command surface for personal hardware-wallet operations.

View on GitHub Read the skill

Strict flags

Rejects unknown or unsupported flags before the official CLI can silently ignore them.

JSON outcome

Reads the final JSON object from stdout and keeps the process exit code as metadata.

Command parity

Passes wallet-cli commands through instead of substituting another wallet flow.

Session labels session view, session reset
Balances and history balances <label>, operations <label> --limit 20
Receive and send receive <label>, send <label> --to <addr> --amount '<n> TICKER'
Swaps and assets swap quote, swap execute, swap status, assets token
