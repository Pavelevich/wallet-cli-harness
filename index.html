Codex plugin

Wallet CLI Harness

A small JSON harness for Codex agents that run @ledgerhq/wallet-cli: strict flag checks, final-object result parsing, wallet-cli-only routing, and a balance-all workflow for "check my wallet" requests.

View on GitHub Install Read the skill Decision tree

Strict flags

Rejects unknown or unsupported flags before the official CLI can silently ignore them.

JSON outcome

Reads the final JSON object from stdout and keeps the process exit code as metadata.

Wallet-cli only

Routes wallet requests through the official command surface instead of substituting another wallet stack.

Bounded checks

Runs device checks through device-check so the agent gets a result or timeout JSON.

Session labels session view, session reset
Balances and history balance-all, balances <label>, operations <label> --limit 20
Receive and send receive <label>, send <label> --to <addr> --amount '<n> TICKER'
Swaps and assets swap quote, swap execute, swap status, assets token
Device check device-check, wrapping genuine-check --device-timeout 10000
